## About The Project



BetChain is a decentralized casino application built on the Monad blockchain. It offers games of chance like "Crash" and classic casino games like Blackjack.

What makes BetChain unique is its innovative savings mechanism. A portion of every bet placed, regardless of the outcome, is automatically deposited into a personal savings vault for the player. This vault also generates a yield, meaning your savings grow over time. It's a casino where you build wealth even when you're not winning.

This project was developed as part of a hackathon, showcasing the power and performance of the Monad network for building high-throughput, engaging, and financially innovative dApps.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have `npm` installed.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your-repo/betchain.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

After starting the development server, connect your Web3 wallet (e.g., MetaMask) and make sure it is configured for the Monad Testnet. You can then navigate to the game lobby, choose a game, and start playing.

-   **/lobby**: The main entry point to see available games.
-   **/crash**: The "El Pollito" crash game.
-   **/blackjack**: A classic game of Blackjack.
-   **/vault**: View and manage your automatic savings.
